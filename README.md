# Odin-recipes

## Description:

This is an introductory project from The Odin Project curriculum focused on HTML fundamentals.

The project consists of building a basic recipe website with a main index page that contains links to several recipe pages.

The skills showcased in the project are understanding of html layout and the application to connected different pages.
